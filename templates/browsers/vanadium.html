Vanadium

Vanadium is possibly the most secure browser out there, building on Chromium's strong security features with additional hardening and privacy enhancements.

Security Features

JIT Compilation Disabled

Vanadium disables Just-In-Time (JIT) compilation by default, reducing the attack surface for vulnerabilities.

Enhanced Exploit Mitigations

Implements stronger Stack Smashing Protection (SSP) and type-based Control Flow Integrity (CFI).

Sandbox and Isolation

Robust Sandboxing

Leverages Chromium's sandboxing technology, isolating each site into its own sandbox.

WebView Integration

The Vanadium rendering engine is used for the WebView component in GrapheneOS, benefiting other apps.

Privacy Features

Enhanced Privacy

Implements enhanced fingerprinting resistance and stricter referrer policies to protect user privacy.

Open Source

Vanadium is open source, allowing for community review and auditing of its codebase, enhancing security through transparency.

Threat Model

Designed to protect against browser exploits, side-channel attacks, cross-site scripting, user tracking, and malicious extensions.
